# 父组件根据子组件数据更新动态更新
## 用到$emit
#### 父组件
```js
    //父组件
    methods:{
        函数名(val){
            console.log(val)
        }
    }

    //子组件上写@自定义事件名='函数名'
    <Child @自定义事件名='函数名'>
```
#### 子组件
```js
    //子组件触发
    this.$emit('自定义事件名'，'传递的数据')
```
## 用到$on和$emit
#### 父组件
```js
    //父组件
    this.$on('自定义事件名',(val)=>{
        console.log(val)
        //val就是传递的数据
    })
```
#### 子组件
```js
    //子组件触发
    this.$parent.$emit('自定义事件名'，'传递的数据')
```
